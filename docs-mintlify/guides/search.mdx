# Search Functionality

## Quick Search

Find products instantly:

1. Click the search icon (🔍) in top bar
2. Type product name, SKU, or keyword
3. Results appear in real-time
4. Click result to view details

### Search Syntax

**Exact match:**
```
"Organic Banana"
```

**Partial match:**
```
banana
```

**Multiple keywords:**
```
organic banana fresh
```

## Saved Searches

Save frequently used searches:

### Create Saved Search

1. Perform search with filters
2. Click **⭐ Save Search**
3. Enter search name
4. Choose private or shared
5. Save

### Manage Saved Searches

Access from **Saved Searches** in sidebar:
- View all saved searches
- Click to execute
- Edit parameters
- Delete old searches
- Share with team

5. Set price alert if needed

## Results Management

### View Options

Choose how to display results:

**Grid View**
- Thumbnail images
- Product name
- Quick price comparison

**List View**
- Detailed information
- All columns visible
- Easier to scan

**Compact View**
- Minimal details
- More items per page
- Quick scrolling

### Sort Options

Organize results:
- **Relevance**: Most relevant first (default)
- **Price - Low to High**: Cheapest first
- **Price - High to Low**: Most expensive first
- **Recently Updated**: Latest data first
- **Availability**: In stock first
- **Popularity**: Most tracked first

### Items Per Page

Show 10, 25, 50, or 100 items per page

## Advanced Search Operators

### Boolean Operators

```
AND: "banana" AND "organic"
OR: "apple" OR "mango"
NOT: "banana" NOT "organic"
```

### Wildcard Search

```
"ban*" matches: banana, bananas, bananaCrisps
"*potato" matches: potato, sweetpotato, carrot
```

### Field-Specific Search

```
category:fruits
platform:zepto
price:<50
stock:instock
```

## Troubleshooting Search

### No Results Found

**Try:**
- Check spelling
- Use broader search terms
- Remove filters
- Check date range
- Verify region selected

### Too Many Results

**Refine by:**
- Adding category filter
- Setting price range
- Selecting platforms
- Narrowing keywords
- Applying date filter

### Wrong Results

**Solutions:**
- Use exact match with quotes
- Add more keywords
- Exclude irrelevant terms
- Filter by category
