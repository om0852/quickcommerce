# Export & Data Download Guide

## Export Formats

### CSV Export

Best for: Spreadsheet analysis in Excel/Google Sheets

**Included columns:**
- Date
- SKU ID
- Product Name
- Platform
- Price
- Currency
- Stock Status
- URL
- Last Updated

**File size**: Depends on data volume

### JSON Export

Best for: API integration and programmatic use

**Structure:**
```json
{
  "export": {
    "date": "2025-03-17",
    "category": "Fruits & Vegetables",
    "region": "Navi Mumbai — 400706",
    "records": [...]
  }
}
```

### Excel Export

Best for: Advanced analysis with formulas

**Features:**
- Multiple sheets (by platform)
- Pre-formatted columns
- Ready-to-use formulas
- Charts included

## Scheduled Exports

Automatically export data on schedule:

### Setup Recurring Export

1. Go to **Settings** → **Exports**
2. Click **Schedule New Export**
3. Configure:
   - Frequency (Daily, Weekly, Monthly)
   - Time of day
   - Format (CSV, JSON, Excel)
   - Recipients (email)
   - Filters (category, platforms, etc.)
4. Save schedule

### Email Delivery

Files are emailed to configured recipients:
- Automatically on schedule
- Formatted for easy download
- With summary stats

## Troubleshooting

### Export Taking Too Long

**Solutions:**
- Reduce date range
- Reduce number of products
- Export by platform separately
- Try different format

### File Format Issues

**Excel import error?**
- Use UTF-8 encoding
- Check column combinations
- Try CSV format first

**JSON parsing error?**
- Validate JSON format online
- Check for special characters
- Update parser settings

### Missing Data in Export

**Verify:**
- All filters were applied
- Date range is correct
- Platforms are selected
- Data actually exists for period
