# Suggestions Endpoint

This page is for suggestions management reference. See [Search & Suggestions](./search) for complete documentation.

## Quick Reference

- `GET /api/suggestions` - List all suggestions
- `POST /api/suggestions` - Create new suggestion
- `PATCH /api/suggestions` - Update suggestion status
- `DELETE /api/suggestions` - Delete suggestion

## Status Values

- `pending` - New suggestion
- `completed` - Implemented
- `rejected` - Not accepted

See [Search & Suggestions](./search) for detailed documentation.
