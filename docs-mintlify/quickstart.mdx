# Quickstart Guide

Get started tracking prices in just 5 minutes!

## Prerequisites

- A web browser (Chrome, Firefox, Safari, or Edge)
- Internet connection
- Access to QuickCommerce Tracker

## Step 1: Login to Dashboard

Navigate to the QuickCommerce Tracker dashboard and sign in with your credentials.

```
https://tracker.creatosaurus.in
```

## Step 2: Select Your First Category

1. Click on the **Category** dropdown
2. Choose a category (e.g., "Fruits & Vegetables")
3. Select a region from the **Region** dropdown (e.g., "Navi Mumbai — 400706")

## Step 3: Choose Date Range

1. Click the **Date** dropdown
2. Select a date or date range to analyze
3. The data will automatically refresh

## Step 4: Filter by Platform

Select which platforms you want to include:
- **All** - Show all platforms
- **Individual platforms** - Select specific ones (JioMart, Zepto, Blinkit, DMart, Flipkart, Instamart)

You can also use:
- **Hide Similar** - Hide similar/duplicate products
- **Show Missing** - Show products missing from certain platforms

## Step 5: View Your Data

The dashboard now shows:
- **Product list** with images and names
- **SKU information** on the left
- **Platform columns** showing prices and availability for each platform
- **Quick comparison** across all platforms

## Common Questions

**Q: What's the data refresh frequency?**  
A: Data is updated in real-time throughout the day across all platforms.

**Q: Can I track multiple categories?**  
A: Yes! Add multiple tracking sessions or use the bookmark feature.

**Q: How far back is historical data available?**  
A: Historical data is available for the past 3 months.
