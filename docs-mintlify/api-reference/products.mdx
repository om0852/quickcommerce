# Products Endpoint

## List Products

Get a list of tracked products with current prices across all platforms.

```http
GET /products
```

### Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `category` | string | Filter by category (fruits, bakery, dairy) |
| `region` | string | Filter by region code (e.g., 400706) |
| `platform` | string | Filter by platform (zepto, blinkit, jiomart, dmart, flipkart, instamart) |
| `minPrice` | number | Filter by minimum price |
| `maxPrice` | number | Filter by maximum price |
| `stock` | string | Filter by stock status (in_stock, out_of_stock, limited) |
| `page` | integer | Page number for pagination (default: 1) |
| `limit` | integer | Items per page (default: 50, max: 100) |
| `sort` | string | Sort field (price_asc, price_desc, updated_desc) |
| `search` | string | Search by product name or SKU |

### Example Request

```bash
curl -X GET 'https://api.tracker.creatosaurus.in/v1/products?category=fruits&platform=zepto&limit=10' \
  -H 'Authorization: Bearer YOUR_API_KEY'
```

## Get Product Details

Get detailed information for a specific product.

```http
GET /products/{sku}
```

### Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `sku` | string | Product SKU (required) |
| `includeHistory` | boolean | Include price history (default: false) |
| `historyDays` | integer | Number of days of history (default: 30) |

### Example Request

```bash
curl -X GET 'https://api.tracker.creatosaurus.in/v1/products/SKU001' \
  -H 'Authorization: Bearer YOUR_API_KEY'
```

## Get Price History

Get historical price data for trend analysis.

```http
GET /products/{sku}/history
```

### Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `sku` | string | Product SKU (required) |
| `from` | string | Start date (YYYY-MM-DD) |
| `to` | string | End date (YYYY-MM-DD) |
| `platform` | string | Specific platform (optional) |
| `interval` | string | Aggregation interval (daily, hourly) |

### Example Request

```bash
curl -X GET 'https://api.tracker.creatosaurus.in/v1/products/SKU001/history?from=2025-03-01&to=2025-03-17' \
  -H 'Authorization: Bearer YOUR_API_KEY'
```

## Error Responses

### Product Not Found

```json
{
  "status": "error",
  "error": {
    "code": "NOT_FOUND",
    "message": "Product SKU001 not found"
  }
}
```

### Invalid Parameter

```json
{
  "status": "error",
  "error": {
    "code": "INVALID_PARAMETER",
    "message": "Invalid category: xyz",
    "details": {
      "field": "category",
      "validValues": ["fruits", "bakery", "dairy"]
    }
  }
}
```
