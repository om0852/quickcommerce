# Alerts API

## Create Alert

Create a new price alert programmatically.

```http
POST /alerts
```

### Request Body

```json
{
  "sku": "SKU001",
  "name": "Banana Price Alert",
  "trigger": "price_below",
  "targetPrice": 100,
  "platforms": ["zepto", "blinkit"],
  "notification": "webhook",
  "webhookUrl": "https://yourapp.com/webhook",
  "active": true
}
```

### Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `sku` | string | Product SKU (required) |
| `name` | string | Alert name (optional) |
| `trigger` | string | Trigger type: price_below, price_above, availability_changed (required) |
| `targetPrice` | number | Target price for trigger (required for price triggers) |
| `platforms` | array | Platforms to monitor (required) |
| `notification` | string | Notification type: webhook, email, sms |
| `webhookUrl` | string | Webhook URL if notification is webhook |
| `email` | string | Email if notification is email |
| `phone` | string | Phone if notification is sms |
| `active` | boolean | Activate alert immediately (default: true) |

### Example Request

```bash
curl -X POST 'https://api.tracker.creatosaurus.in/v1/alerts' \
  -H 'Authorization: Bearer YOUR_API_KEY' \
  -H 'Content-Type: application/json' \
  -d '{
    "sku": "SKU001",
    "trigger": "price_below",
    "targetPrice": 100,
    "platforms": ["zepto", "blinkit"],
    "notification": "webhook",
    "webhookUrl": "https://yourapp.com/webhook"
  }'
```

### Example Response

```json
{
  "status": "success",
  "data": {
    "alertId": "ALERT_ABC123",
    "sku": "SKU001",
    "trigger": "price_below",
    "targetPrice": 100,
    "currentPrice": 120,
    "platforms": ["zepto", "blinkit"],
    "notification": "webhook",
    "active": true,
    "createdAt": "2025-03-17T10:30:00Z",
    "updatedAt": "2025-03-17T10:30:00Z"
  }
}
```

## List Alerts

Get all active alerts for your account.

```http
GET /alerts
```

### Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `status` | string | Filter by status: active, triggered, paused |
| `sku` | string | Filter by product SKU |
| `page` | integer | Page number (default: 1) |
| `limit` | integer | Items per page (default: 50) |

### Example Request

```bash
curl -X GET 'https://api.tracker.creatosaurus.in/v1/alerts?status=active&limit=10' \
  -H 'Authorization: Bearer YOUR_API_KEY'
```

### Example Response

```json
{
  "status": "success",
  "data": [
    {
      "alertId": "ALERT_ABC123",
      "sku": "SKU001",
      "product": "Organic Banana",
      "trigger": "price_below",
      "targetPrice": 100,
      "currentPrice": 120,
      "status": "active",
      "triggered": false,
      "lastTriggered": null,
      "createdAt": "2025-03-17T10:30:00Z"
    }
  ],
  "pagination": {
    "page": 1,
    "limit": 10,
    "total": 25
  }
}
```

## Get Alert Details

Get details of a specific alert.

```http
GET /alerts/{alertId}
```

### Example Request

```bash
curl -X GET 'https://api.tracker.creatosaurus.in/v1/alerts/ALERT_ABC123' \
  -H 'Authorization: Bearer YOUR_API_KEY'
```

### Example Response

```json
{
  "status": "success",
  "data": {
    "alertId": "ALERT_ABC123",
    "sku": "SKU001",
    "product": "Organic Banana",
    "trigger": "price_below",
    "targetPrice": 100,
    "currentPrice": 120,
    "platforms": ["zepto", "blinkit"],
    "notification": "webhook",
    "webhookUrl": "https://yourapp.com/webhook",
    "status": "active",
    "createdAt": "2025-03-17T10:30:00Z",
    "history": [
      {
        "date": "2025-03-17T09:00:00Z",
        "price": 119,
        "triggered": false
      }
    ]
  }
}
```

## Update Alert

Modify an existing alert.

```http
PUT /alerts/{alertId}
```

### Request Body

```json
{
  "targetPrice": 95,
  "platforms": ["zepto", "blinkit", "jiomart"],
  "active": true
}
```

### Example Response

```json
{
  "status": "success",
  "data": {
    "alertId": "ALERT_ABC123",
    "targetPrice": 95,
    "platforms": ["zepto", "blinkit", "jiomart"],
    "active": true,
    "updatedAt": "2025-03-17T10:35:00Z"
  }
}
```

## Delete Alert

Delete an alert.

```http
DELETE /alerts/{alertId}
```

### Example Response

```json
{
  "status": "success",
  "message": "Alert deleted successfully"
}
```

## Bulk Operations

### Create Multiple Alerts

```http
POST /alerts/bulk
```

### Request Body

```json
{
  "alerts": [
    {
      "sku": "SKU001",
      "trigger": "price_below",
      "targetPrice": 100,
      "platforms": ["zepto"]
    },
    {
      "sku": "SKU002",
      "trigger": "price_below",
      "targetPrice": 50,
      "platforms": ["blinkit"]
    }
  ]
}
```

### Response

```json
{
  "status": "success",
  "data": [
    { "alertId": "ALERT_001", "status": "created" },
    { "alertId": "ALERT_002", "status": "created" }
  ]
}
```

## Error Responses

### Invalid SKU

```json
{
  "status": "error",
  "error": {
    "code": "INVALID_SKU",
    "message": "SKU SKU001 not found"
  }
}
```

### Alert Already Exists

```json
{
  "status": "error",
  "error": {
    "code": "DUPLICATE_ALERT",
    "message": "An alert already exists for this product"
  }
}
```
