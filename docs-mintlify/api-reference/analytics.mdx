# Analytics API

## Get Analytics Summary

Get aggregated analytics for a category or region.

```http
GET /analytics/summary
```

### Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `category` | string | Filter by category (required) |
| `region` | string | Filter by region (required) |
| `from` | string | Start date (YYYY-MM-DD) |
| `to` | string | End date (YYYY-MM-DD) |
| `platforms` | array | Specific platforms to analyze |

### Example Response

```json
{
  "status": "success",
  "data": {
    "category": "Fruits & Vegetables",
    "region": "Navi Mumbai",
    "period": "2025-03-01 to 2025-03-17",
    "metrics": {
      "avgPrice": 112.50,
      "minPrice": 38,
      "maxPrice": 250,
      "priceVolatility": 4.2,
      "productsTracked": 145,
      "platformsActive": 6,
      "availabilityRate": 94.5
    },
    "trends": {
      "weekOverWeek": -2.3,
      "monthOverMonth": 1.5,
      "direction": "downward"
    }
  }
}
```

## Get Price Trends

Analyze price trends over time.

```http
GET /analytics/trends
```

### Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `category` | string | Filter by category |
| `product` | string | Specific product SKU |
| `from` | string | Start date |
| `to` | string | End date |
| `interval` | string | daily, weekly, monthly |

### Example Response

```json
{
  "status": "success",
  "data": {
    "trends": [
      {
        "date": "2025-03-17",
        "price": 116.60,
        "minPrice": 108,
        "maxPrice": 120,
        "volume": 1250,
        "volatility": 3.8
      }
    ]
  }
}
```

## Compare Products

Compare analytics for multiple products.

```http
GET /analytics/compare
```

### Parameters

| Parameter | Type | Description |
|-----------|------|-------------|
| `skus` | array | SKUs to compare (required) |
| `from` | string | Start date |
| `to` | string | End date |

### Example Response

```json
{
  "status": "success",
  "data": {
    "comparison": [
      {
        "sku": "SKU001",
        "name": "Organic Banana",
        "avgPrice": 116.60,
        "priceChange": -2.3,
        "mostExpensive": "zepto",
        "cheapest": "dmart"
      }
    ]
  }
}
```
